## Installation

Install Mama Recipe API with npm :

-   Clone the Repo

```
  git clone https://github.com/ariffebri22/RecipeAPIV2.2.git
```

-   Go To Folder Repo

```
  cd RecipeAPIV2.2
```

-   Install Module

```
  npm install
```

-   Setup .env

```
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_PORT=
SECRET_KEY=
CD_NAME=
CD_API_KEY=
CD_API_SECRET=
EMAIL=
PASS=
```

-   Type `npm run start` To Start Production
